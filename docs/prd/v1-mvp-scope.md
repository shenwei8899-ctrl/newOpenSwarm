# OpenSwarm SaaS V1 MVP Scope

## In scope

- Employee-market home page
- Team creation flow
- Team member selection
- Skill catalog and team-level employee skill assignment
- Team workspace shell
- Group discussion sessions
- Discussion summaries
- Task creation from discussion
- Single-employee task execution
- Artifact listing

## Primary user flow

1. browse standard employees on the home page
2. create a team
3. confirm team members
4. assign employee skills
5. enter the team workspace
6. run discussion and create tasks from the discussion context

## Out of scope

- Autonomous employee handoff
- Complex workflow automation
- Billing center
- Full enterprise RBAC
- Advanced realtime collaboration
